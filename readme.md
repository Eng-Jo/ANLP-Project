# CV Analyzer - Student Resume Analysis Tool

A Python GUI application that analyzes student CVs/resumes and provides scoring and improvement suggestions.

## Features

- ✅ Support for PDF and DOCX files
- ✅ Automatic scoring (0-100)
- ✅ Section completeness check
- ✅ Technical skills detection
- ✅ Action verbs analysis
- ✅ Contact information extraction
- ✅ Length analysis
- ✅ Detailed improvement suggestionss
- ✅ User-friendly GUI interface
